# Jobmagica Website

Jobmagica Website
=================

This repository contains UI screens and assets for the Jobmagica project (login, dashboard, resume upload, and related screens).

Quick start
-----------
- Open the HTML files in the `stitch_jobmagica_login` folder in a browser to view the mockups/screens.

Project structure
-----------------
- `stitch_jobmagica_login/` — main folder containing HTML mockups and images.

Notes
-----
- This repo is a static collection of UI files (HTML + images). There is no build step required—just open the HTML files.
